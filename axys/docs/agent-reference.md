# AXYS 에이전트 역할 카드

> 10명의 AI 에이전트 상세 안내

## 조직도

```
                        ┌─────────────┐
                        │    교장      │
                        │  Principal   │
                        │ [final]      │
                        └──────┬──────┘
                               │
                        ┌──────┴──────┐
                        │    교감      │
                        │Vice Principal│
                        │ [final]      │
                        └──────┬──────┘
                               │
              ┌────────────────┼────────────────┬──────────────┐
              │                │                │              │
       ┌──────┴──────┐ ┌──────┴──────┐ ┌──────┴──────┐ ┌────┴─────┐
       │  교무부장    │ │  연구부장    │ │  학년부장    │ │ 행정실장  │
       │Academic Head │ │Research Head │ │ Grade Head   │ │Admin Head │
       │ [recommend]  │ │ [recommend]  │ │ [execute]    │ │[recommend]│
       └──────┬──────┘ └─────────────┘ └──────┬──────┘ └──────────┘
              │                                │
       ┌──────┴──────┐                  ┌──────┴──────┐
       │  교과교사    │                  │  담임교사    │
       │Subject Tchr  │                  │Homeroom Tchr │
       │ [advise]     │                  │ [execute]    │
       └─────────────┘                  └──────┬──────┘
                                               │
       ┌─────────────┐                  ┌──────┴──────┐
       │  상담교사    │                  │  학생대표    │
       │ Counselor    │                  │Student Rep   │
       │ [advise]     │                  │ [advise]     │
       └─────────────┘                  └─────────────┘
        (교감 직속)
```

범례: `[final]` 최종 결정 / `[recommend]` 추천 및 기획 / `[execute]` 실행 / `[advise]` 자문

## 1. 교장 (Principal)

| 항목 | 내용 |
|------|------|
| **이름** | 교장 / Principal |
| **ID** | `principal` |
| **역할 범주** | leadership |
| **의사결정 권한** | final (최종 승인) |
| **보고 대상** | 없음 (최상위) |
| **감독 대상** | 교감 |

**핵심 역할 요약**

학교 경영의 최종 책임자로서 교육 비전을 수립하고 학교 발전의 큰 방향을 제시한다. 모든 시나리오의 최종 의사결정자로서 승인 또는 반려 권한을 행사하며, 교육청과 지역사회와의 대외 관계에서 학교를 대표한다.

**참여 시나리오**

모든 시나리오에 최종 승인자로 참여

---

## 2. 교감 (Vice Principal)

| 항목 | 내용 |
|------|------|
| **이름** | 교감 / Vice Principal |
| **ID** | `vice-principal` |
| **역할 범주** | leadership |
| **의사결정 권한** | final (일상 운영 결정) |
| **보고 대상** | 교장 |
| **감독 대상** | 교무부장, 연구부장, 학년부장, 행정실장 |

**핵심 역할 요약**

학교 운영의 실무 총괄자로서 교장을 보좌하며 교무 전반을 관리한다. 각 부장교사의 업무를 조율하고 통합하는 허브 역할을 수행하며, 일상적 학교 운영 사항에 대해서는 자체적으로 의사결정을 수행한다.

**참여 시나리오**

- 학교 행사 기획 (주관)
- 학교 안전 계획 (주관)
- 학교 자체 평가 (주관)
- 학부모 소통 (주관)
- 학기 전환 업무 (주관)
- 위기 대응 매뉴얼 (주관)
- 교육과정 편성, 예산 편성, 교원 연수 등 (참여)

## 3. 교무부장 (Academic Affairs Head)

| 항목 | 내용 |
|------|------|
| **이름** | 교무부장 / Academic Affairs Head |
| **ID** | `academic-head` |
| **역할 범주** | management |
| **의사결정 권한** | recommend (기획 및 추천) |
| **보고 대상** | 교감 |
| **감독 대상** | 담임교사, 교과교사 |

**핵심 역할 요약**

교육과정 편성과 운영의 실무 책임자로서 학교 교육 활동의 중심축을 담당한다. 수업 시간표를 편성하고 학사 일정을 관리하며 성적 처리를 총괄한다. 교과 운영을 지원하고 학기 전환기 업무를 주도적으로 기획한다.

**참여 시나리오**

- 교육과정 편성 (주관)
- 학기 전환 업무, 학교 자체 평가 (참여)
- 학교 행사 기획, 예산 편성 (참여)

## 4. 연구부장 (Research & Development Head)

| **ID** | `research-head` |

| **의사결정 권한** | recommend (기획 및 추천) |
| **보고 대상** | 교감 |
| **감독 대상** | 없음 |

**핵심 역할 요약**

## 5. 학년부장 (Grade Level Head)

| 항목 | 내용 |
|------|------|
| **이름** | 학년부장 / Grade Level Head |
| **ID** | `grade-head` |
| **역할 범주** | management |
| **의사결정 권한** | execute (실행) |
| **보고 대상** | 교감 |
| **감독 대상** | 담임교사 |

**핵심 역할 요약**

해당 학년의 학생 생활지도를 총괄하는 실무 책임자이다. 학년 단위의 학생 생활 규정을 운영하고 학년 행사를 기획하며, 소속 담임교사들을 지원하고 업무를 조율한다. 학년 내 학생 문제 발생 시 초기 대응을 지휘한다.

**참여 시나리오**

- 학생 생활지도 (주관)
- 학기 전환 업무, 위기 대응 매뉴얼 (참여)
- 학교 행사 기획, 교육과정 편성 (참여)

## 6. 담임교사 (Homeroom Teacher)

| 항목 | 내용 |
|------|------|
| **이름** | 담임교사 / Homeroom Teacher |
| **ID** | `homeroom-teacher` |
| **역할 범주** | teaching |
| **의사결정 권한** | execute (실행) |
| **보고 대상** | 학년부장 |
| **감독 대상** | 학생대표 |

**핵심 역할 요약**

학급 경영의 책임자로서 학생들의 학교생활 전반을 돌본다. 학생 출결을 관리하고 학급 생활지도를 수행하며, 학부모와의 소통 창구 역할을 한다. 학생 개개인의 특성을 파악하여 맞춤형 지도를 제공하는 가장 가까운 교육자이다.

**참여 시나리오**

- 학생 생활지도, 학부모 소통, 학기 전환 업무 (참여)
- 학교 행사 기획 (참여)

## 의사결정 권한 체계

| 권한 수준 | 설명 | 해당 에이전트 | AI 모델 |
|----------|------|-------------|---------|
| **final** | 최종 승인 권한 | 교장, 교감 | opus |
| **recommend** | 기획 및 추천 권한 | 교무부장, 연구부장, 행정실장 | sonnet |
| **execute** | 결정 사항 실행 | 학년부장, 담임교사 | sonnet |
| **advise** | 자문 및 의견 제시 | 교과교사, 상담교사, 학생대표 | haiku |
