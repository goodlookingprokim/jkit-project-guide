# AXYS 교육과정 편성 시나리오

**Agent**: Use the `axys-academic-head` agent for this command.
**Language**: Respond in **Korean** for all user-facing messages.

## Purpose

테스트학교의 교육과정을 편성하고 운영 계획을 수립하는 핵심 업무입니다.
국가 교육과정 기준과 교육청 지침을 기반으로 학교 특색과 학생 실태를 반영하여
학년별, 교과별 수업 계획을 체계적으로 수립합니다.

## Input

`$ARGUMENTS` -- 편성 관련 추가 지시사항이나 참고 자료.
비어 있으면 기본 교육과정 편성 절차를 시작합니다.

## Implementation Flow

### Phase 1: 킥오프

1. 사용자에게 인사하고 교육과정 편성 시작을 안내합니다:
   "테스트학교의 교육과정 편성을 시작하겠습니다."

2. 기초 자료를 확인합니다:
   - 교육청 교육과정 편성 지침이 있는지 확인
   - 전년도 교육과정 편성표가 있는지 확인
   - 올해 중점 교육활동 방향 확인

3. 사용자에게 질문합니다:
   - "교육청 편성 지침 파일이 있으시면 공유해 주세요."
   - "올해 특별히 강화하고 싶은 교육 영역이 있나요?"
   - "학급 수와 학년 구성을 알려주세요."

4. 교장(axys-principal)의 교육 비전 방향을 참고합니다.

### Phase 2: 역할별 작업

**교무부장(본 에이전트)의 작업:**
1. 교육청 지침을 분석하여 편성 기준안을 작성합니다
2. 학년별 총 수업 시수와 교과별 배당 시수를 산출합니다
3. 시간표 초안을 편성합니다
4. 각 교과 및 학년의 의견을 종합합니다

**다른 에이전트와의 협업:**
- Delegate to `axys-research-head` via Task(): 교육청 지침 분석 및 혁신 사례 조사
- Delegate to `axys-subject-teacher` via Task(): 교과별 운영 계획서 및 필요 시수 산출
- Delegate to `axys-grade-head` via Task(): 학년별 특성 및 요구사항 수렴

**중간 산출물 제시:**
- 편성 기준안 초안을 사용자에게 보여줍니다
- 시간표 초안을 사용자에게 보여줍니다
- 수정이 필요한 부분이 있는지 확인합니다

### Phase 3: 통합 검증

1. 모든 교과별 운영 계획서와 학년별 의견을 취합합니다
2. 교육과정 편성표, 시간표, 교과별 운영 계획서를 통합합니다
3. 법정 기준 충족 여부를 점검합니다 (수업일수, 시수, 필수교과 등)

**검토 요청:**
- Delegate to `axys-vice-principal` via Task(): 편성안의 적합성, 실현 가능성, 법적 요건 충족 여부 검토
- 교과 간 시수 균형, 교사별 수업 부담 형평성 확인

**최종 승인:**
- 교감의 검토가 완료되면 교장(axys-principal)에게 최종 승인을 요청합니다
- 승인되면 사용자에게 확정된 교육과정을 안내합니다

4. 최종 산출물을 정리하여 제시합니다:
   - 교육과정 편성표
   - 시간표 초안
   - 교과별 운영 계획서

사용자에게 안내합니다:
"교육과정 편성이 완료되었습니다. 확인해 보시고 수정이 필요한 부분이 있으면 말씀해 주세요."
